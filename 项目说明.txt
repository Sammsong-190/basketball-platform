篮球用品电商社交平台 - 项目结构说明

已创建的核心文件：

1. 数据库模型
   - prisma/schema.prisma - 完整的数据库模型定义

2. 工具库 (lib/)
   - prisma.ts - Prisma客户端
   - auth.ts - JWT认证工具
   - middleware.ts - 认证和权限中间件
   - utils.ts - 工具函数

3. API路由 (app/api/)
   - auth/register/route.ts - 用户注册
   - auth/login/route.ts - 用户登录
   - users/profile/route.ts - 个人信息管理
   - users/favorites/route.ts - 收藏管理
   - users/my-comments/route.ts - 评论管理
   - users/my-orders/route.ts - 订单管理
   - users/my-posts/route.ts - 发帖记录
   - users/expenses/route.ts - 支出信息
   - users/incomes/route.ts - 收入信息（卖家）
   - posts/route.ts - 帖子列表和创建
   - posts/[id]/route.ts - 帖子详情、编辑、删除
   - posts/[id]/like/route.ts - 帖子点赞
   - comments/route.ts - 评论功能
   - products/route.ts - 商品管理
   - categories/route.ts - 分类管理
   - cart/route.ts - 购物车管理
   - orders/route.ts - 订单管理

4. 配置文件
   - package.json - 项目依赖和脚本
   - tsconfig.json - TypeScript配置
   - next.config.js - Next.js配置
   - tailwind.config.ts - Tailwind CSS配置
   - postcss.config.js - PostCSS配置

5. 页面文件
   - app/layout.tsx - 根布局
   - app/page.tsx - 首页
   - app/globals.css - 全局样式

快速开始步骤：

1. 安装依赖：
   npm install

2. 初始化数据库：
   npm run db:generate
   npm run db:push

3. 启动开发服务器：
   npm run dev

4. 访问：
   http://localhost:3000

注意：还需要创建以下API路由文件：
- app/api/products/[id]/route.ts - 商品详情和编辑
- app/api/orders/[id]/route.ts - 订单详情和状态更新
- app/api/refunds/route.ts - 退换货管理
- app/api/complaints/route.ts - 投诉建议
- app/api/admin/* - 系统管理相关API

以及前端页面组件等。
